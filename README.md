# St.-PET-ER (Beta name as joke)

Naa ray daghan sa youtube mga top extensions,

mao ni ang mga gamit kaayo sa general webdev, para nako:

- Prettier
- Live Server
- Auto Close Tag
- Auto Rename
- CSS Colorizer
- HTML CSS Support (by ecmel)
- Intellisense for CSS (by Zignd)
- Better Comments
